# jivan                                                       

urls generator of a link with many subdirectories.            
                                                              
- URL generator                                               
```sh                                                         
$ ./jivan -u https://www.dominio.com/pagina1/pagina2/pagina3/f
ile.html -w text                                              
```                                                           
- URL generator and verify                                    
```sh                                                         
$ ./jivan -u https://www.dominio.com/pagina1/pagina2/pagina3/f
ile.html -w text -v True -m GET                               
```
